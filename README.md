Smile School Website

Project Overview

This project is a responsive website for Smile School, an online platform offering tutorials on smiling techniques and courses by professionals.

The website features:

- A clean and accessible layout with semantic HTML5 elements (<header>, <main>, <footer>)
- Navigation menu with links to Courses, Pricing, and Login
- Sections showcasing featured instructors with images and achievements
- Testimonials from users
- A footer with social media icons and copyright info

Technologies Used
HTML5
- CSS (to be added separately)
- Font Awesome for social media icons

Structure

- index.html: Main homepage of the Smile School website
- Images are located in the images folder (referenced in the HTML)

How to Use

1. Open index.html in any modern web browser.
2. Navigate through the site using the menu.
3. The layout is responsive and designed to work on various screen sizes.
